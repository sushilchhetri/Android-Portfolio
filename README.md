# 🚀 Featured Project: Galaxy Hotspot Chat
**Status: Live on Play Store | 500+ Downloads | Verified Offline Connectivity**

Galaxy Hotspot Chat is a high-performance communication tool that allows users to chat without internet using Android Socket programming and Wi-Fi Hotspots. It is a showcase of my ability to handle complex low-level networking.

<a href='https://play.google.com/store/apps/details?id=com.ConnectChat'><img alt='Get it on Google Play' src='https://play.google.com/intl/en_us/badges/images/generic/en_badge_web_generic.png' height='60px'/></a>

### Engineering Highlights:
- **P2P Networking**: Custom implementation of Socket Programming to facilitate real-time group chat over local hotspots.
- **Scalable Architecture**: Built with **MVVM** and **Hilt** to ensure the app remains stable under heavy local traffic.
- **User Growth**: Successfully scaled to **500+ organic users** with positive feedback on connectivity speed.
